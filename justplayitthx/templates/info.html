justplayitthx FAQ

1 .why is it kinda slow.. ish?

my wifi isnt exactly the best thing, the server also isnt but its fine for light sites like these. this project is fully selfhosted (with the help of cf tunnels cause i dont have port forwarding). one day i might get a VPS but i have no idea when and if it will have enough power/storage to run this site.

2. this doesnt work on [insert old broswer]!!!1111

some old browsers might not work, because this website still uses modern frameworks (like javascript). i will try to make it support more old browsers, but the current focus for me is to make a low memory and distraction free youtube experience.

3. what do i need to have installed?

basically nothing. this is a purely web experience and doesnt need any external software installation, except (obviously) a internet connection and a supported browser. however, some browsers may be too old for jpit. but most semi-new browser are supported on, example: XP and 7.

4. what is this site even for???

this is a low-memory, distraction free and minimal youtube client, basically it downloads a video on our server then your shitty machine can play it without hassle and gui distractions.
